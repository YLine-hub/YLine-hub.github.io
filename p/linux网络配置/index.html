虚拟网络

介绍

（1）windows与虚拟网络联通

window下有一个网卡为 “VMware Network Adapter VMnet8” ，是用于与虚拟机进行通讯。 window可以用ipconfig查看ip地址，linux可以用ifconfig查看ip地址。 可以看到这两个网络是同一网段。

（2）windows与internet连接

  • 通过windows下的真实网卡，经历路由器网关，跟互联网进行交互。

查看默认网关 

1

ip route show

# 查看网卡配置文件
$ cat /etc/sysconfig/network-scripts/ifcfg-ens33
TYPE="Ethernet"         # 网卡类型，Ethernet表示以太网
PROXY_METHOD="none"     # 代理方式：关闭状态
BROWSER_ONLY="no"       # 只是浏览器（yes|no）
BOOTPROTO="dhcp"        # 设置网卡获得ip地址的方式（static|dhcp|none）
DEFROUTE="yes"          # 设置为默认路由（yes|no）
IPV4_FAILURE_FATAL="no" # 是否开启IPV4致命错误检测（yes|no）
IPV6INIT="yes"          # IPV6是否自动初始化
IPV6_AUTOCONF="yes"     # IPV6是否自动配置
IPV6_DEFROUTE="yes"     # IPV6是否可以为默认路由
IPV6_FAILURE_FATAL="no" # 是否开启IPV6致命错误检测
IPV6_ADDR_GEN_MODE="stable-privacy" # IPV6地址生成模型
NAME="ens33"            # 网卡物理设备名称
UUID="8b1f9fd5-9d4b-46f7-ba3f-0ff3f862e0c2" # UUID识别码
DEVICE="ens33"          # 网卡设备名称
ONBOOT="yes"            # 开机自启

配置静态ip

  • 打开文件ifcfg-ens33
1

vi /etc/sysconfig/network-scripts/ifcfg-ens33

  • 配置参数

1
2
3
4
5

BOOTPROTO="static"
IPADDR=192.168.172.100
# 网关和DNS1相同即可
GATWAY=192.168.172.2
DNS1=192.168.172.2
  • 重启网络
1

systemctl restart network
