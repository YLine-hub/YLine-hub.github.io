解决 Windows 错误 0x803f7001

当你在运行 Microsoft Windows 非核心版本的计算机上遇到错误代码 0x803f7001 时，通常表示系统无法找到有效的许可证来激活 Windows。以下是解决此问题的步骤。

如果没有出现这个报错则不用进行这一步

修改注册表

  1. 按下 Win + R 打开“运行”，输入 regedit 后回车。

  2. 导航到 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SoftwareProtectionPlatform。

  3. 找到 SkipRearm，将其值从 0 改为 1，然后重启电脑。

激活系统

  1. 右击“开始”菜单，选择“Windows PowerShell（管理员）”

  2. 输入以下命令

1
2
3

slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
slmgr /skms kms.03k.org
slmgr /ato
