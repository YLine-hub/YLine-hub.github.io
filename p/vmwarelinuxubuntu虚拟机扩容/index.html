# 1. 先使用 parted 扩展物理分区 /dev/sda3（让它占用磁盘剩余未分配空间）
parted /dev/sda

# 进入 parted 交互模式后执行：
(parted) resizepart 3  # 扩展第 3 个分区（/dev/sda3）
(parted) 100%           # 把分区扩展到磁盘末尾（占用全部未分配空间）
(parted) quit           # 退出

# 2. 刷新物理卷（让 LVM 识别 /dev/sda3 扩展后的容量）
sudo pvresize /dev/sda3

查看未分配的空间

1

vgdisplay ubuntu-vg

操作扩容

直接将卷组的全部空闲空间分配给根目录逻辑卷

1

lvextend -l +100%FREE /dev/mapper/ubuntu--vg-ubuntu--lv

扩展 ext4 文件系统

1

resize2fs /dev/mapper/ubuntu--vg-ubuntu--lv

查看是否成功

1

df -h
