Shell编程

介绍

  • Shell 是一个命令行解释器，它为用户提供了一个向Linux内核发送请求以便运行程序的界面系统级程序，用户可以用Shell来启动、挂起、停止甚至是编写一些程序。

Shell变量

介绍

  • Shell变量：系统变量 和 用户自定义变量
  • 系统变量：$HOME，$PWD，$SHELL，$USER
  • set指令可查看所有系统变量
  • 自定义变量：变量=值
  • 撤销变量：unset 变量
  • 声明静态变量：readonly 变量（不能unset）

变量定义规则

  • 只包含字母、数字和下划线，不能以数字开头
  • 避免使用 Shell 关键字，如if、then、else、fi、for、while等
  • 使用大写字母表示常量
  • 避免使用特殊符号和空格

设置环境变量

介绍

  • linux上所有的环境变量都配置在/etc/profile中
  • export 变量=值来配置环境变量
  • 配置完毕后，通过source /etc/profile，重新加载配置文件来使用

  • 输出测试

echo "${LINUX_OWNER}"

位置参数变量

介绍

在我们执行shell脚本的时候，可以在后面拼接参数，进行传参，例：./test.sh hello world。
shell脚本有对应的参数变量来接收这些参数。

  • $n （n为数字）

    • 以空格来切分命令，将上面命令变成[./test.sh, hello, world]，通过$0, $1, $2来获取参数
    • 如果个数达到10位数以上，需要用大括号括住数字${10}

  • $*

    • 以一个单字符串显示所有向脚本传递的参数。
    • 如"$*“用「"」括起来的情况、以”$1 $2 … $n"的形式输出所有参数。
    • 获取全部参数，会将传入的参数hello world视为一个整体

  • $@

    • 与$*相同，但是使用时加引号，并在引号中返回每个参数。
    • 如"$@“用「"」括起来的情况、以”$1" “$2” … “$n” 的形式输出所有参数。
    • 获取全部参数，与$*不同，会将hello和world区分为个体

  • $#

    • 统计传入参数的个数

预定义变量

介绍

shell本身有定义好的变量提供用户使用。

  • $$

    • 获取当前进行的PID

  • $!

    • 获取后台运行的最后一个进程的进程号

  • $?

    • 最后一次执行的命令的返回状态（0=正常，非0=异常）

关联数组

Bash 支持关联数组，可以使用任意的字符串、或者整数作为下标来访问数组元素。

  • 格式
    • -A 选项就是用于声明一个关联数组
    • 关联数组的键是唯一的。

declare -A site

site["taobao"]="www.taobao.com"

echo ${site["baidu"]}
