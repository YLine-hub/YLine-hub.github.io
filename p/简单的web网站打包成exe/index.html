Nativefier

简单的web网站打包成exe

2025-10-14

前置环境

  • nodejs:nodejs-download

  • nativefier插件

  • 网站链接

打包过程

安装nodejs

  • 运行安装程序

  • 一直点击next，可以改一下安装路径，其他不用修改

  • 测试是否安装

  • 打开cmd，输入命令查看版本

1

npm -v

安装插件

  • 用npm安装插件
1

npm install nativefier -g
  • 等待安装完成

打包 

1

nativefier --name "ciruitjs1" "http://www.haoji.cc/"

问题二：修改hosts文件没有权限

  • 给hosts文件添加写入权限

  • 右键hosts文件，选择属性

  • 点击安全

  • 点击编辑

  • 选择Users，点击最上面的选项给予全部权限，再点击应用

  • 修改完权限以后，修改hosts文件

重新打包

  • 再次执行打包命令
1

nativefier --name "ciruitjs1" "http://www.haoji.cc/"
  • 等待打包完成

  • 打包完成，位置在打包的目录下

  • 进入目录后，可以看到打包后的程序

  • 双击运行
